# Images Folder

## Required Images

Please add the following images to this folder:

### 1. giuseppe-funaro.jpg
- Your professional headshot
- This will appear on the About page in the bio section
- Recommended size: 1000x1000px minimum
- Format: JPG or PNG

### 2. logo.png (Optional)
- Company logo with transparent background
- This will appear in the navigation bar next to the text logo
- Recommended size: 200x200px (will be displayed at 64px height)
- Format: PNG with transparency

## How to Add Images

1. Save your headshot as `giuseppe-funaro.jpg` in this folder
2. Save your logo as `logo.png` in this folder (optional)
3. Rebuild the project: `npm run build`
4. Push to GitHub to deploy

The website is configured to work with or without the logo image. If the logo file is not found, only the text logo will display.
